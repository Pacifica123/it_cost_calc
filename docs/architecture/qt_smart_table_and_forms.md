# Qt smart table and forms

## Назначение

Патч создаёт общий строительный блок для будущих Qt-экранов с данными.
Цель — не плодить разные таблицы, кнопки и формы в каждом разделе.

## Правила UI

- основное действие таблицы видно сразу;
- вторичные действия живут в `Ещё` или context-menu;
- пустая таблица показывает короткое состояние;
- длинные объяснения хранятся в tooltip/help;
- label и placeholder не длиннее 10 слов;
- формы строятся по `FieldSpec` и проверяются до сохранения.

## Компоненты

- `SmartTable` — компактная таблица с empty state и CRUD-действиями;
- `RecordFormDialog` — единый диалог записи;
- `FieldSpec` — схема поля с валидатором и tooltip-help;
- `RowTableModel` — import-safe модель с CRUD-хелперами.
