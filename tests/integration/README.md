# Интеграционные тесты

В этом каталоге лежат не GUI-тесты, а сквозные сценарии поверх нескольких слоёв приложения сразу.

Здесь проверяются:
- загрузка демонстрационного набора в JSON-backed runtime-репозиторий;
- подготовка сводки затрат и экспорт итогового отчёта в CSV;
- запуск AHP-анализа на эталонном наборе конфигураций и сохранение отчёта в JSON;
- построение каталога оборудования из example-снимков.

Такие тесты нужны как мост между unit-проверками и ручной проверкой через GUI.
