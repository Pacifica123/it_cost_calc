# Qt export screen

Патч `qt-export-screen` переносит корневой раздел `Экспорт` в новый Qt-контур.

Экран сохраняет принцип миграции: один основной сценарий, минимум постоянных
элементов и подробности в раскрываемых блоках. Главный сценарий — выбрать формат
и нажать `Собрать отчёт`.

## Что умеет экран

- собирает полный DecisionReport в JSON, Markdown и CSV;
- экспортирует отдельный CSV затрат;
- формирует частичные Markdown-фрагменты по затратам и NPV;
- показывает краткую сводку данных перед экспортом;
- показывает последние созданные файлы.

## Ограничения этапа

Маршрут ПО/ТО ещё не перенесён в Qt, поэтому частичный экспорт аналитики ПО/ТО
создаёт контекстный Markdown-маркер. Полные GA/AHP/Pareto/Hybrid данные будут
подключены на последующих патчах маршрута ПО/ТО.
