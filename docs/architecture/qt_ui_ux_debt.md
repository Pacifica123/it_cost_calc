# Qt UI UX debt

Этот файл фиксирует решения, которые допустимы на этапе миграции, но требуют общего polish-прохода перед переключением Qt UI по умолчанию.

## Редактор компонентов

Текущий вид после патча `qt-component-editor-form-polish` принимается как рабочий, но не финальный.

Долг:

- верхняя панель всё ещё немного шумная;
- форма занимает много пустого пространства;
- `ⓘ` рядом с каждым блоком может раздражать;
- блоки формы выглядят одинаково;
- при общем polish нужно усилить визуальный приоритет разделов.

Решение: не блокировать перенос остальных экранов. Вернуться к редактору компонентов после появления нескольких Qt-экранов и единого визуального сравнения.
