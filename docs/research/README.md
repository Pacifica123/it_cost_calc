# Исследовательские материалы

Этот раздел нужен для того, чтобы не смешивать:

- инженерную документацию репозитория;
- академические raw-материалы диплома;
- исследовательские и методические заметки по алгоритмам и вспомогательным инструментам.

## Что здесь находится

### `ahp.md`
Краткое описание AHP-направления в проекте:
- что анализируется;
- какие данные подаются на вход;
- какой результат получается на выходе;
- где лежит код и примеры данных.

### `criteria_importance.md`
Описание модуля анализа важности критериев:
- структура кейса;
- идея отношений важности;
- тип результата;
- связь с задачей обоснования выбора ИТ-решения.

### `parsing.md`
Краткая исследовательская сводка по парсеру каталога оборудования:
- зачем он нужен;
- почему не считается частью ядра продукта;
- какие ограничения были выявлены на практике.

## Чем этот раздел отличается от `docs/thesis/`

- `docs/research/` — это человекочитаемые инженерно-исследовательские заметки для проекта;
- `docs/thesis/raw/` — это исходные дипломные документы, презентации и таблицы.

Именно такое разделение делает репозиторий более аккуратным и понятным.
