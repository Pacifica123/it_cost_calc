# Qt main navigation

## Назначение

Патч добавляет главный каркас нового Qt-интерфейса. Он не переносит бизнес-экраны и не переключает основной запуск приложения. Старый Tkinter UI остаётся рабочим, а новый Qt-контур получает понятную навигацию для дальнейших патчей.

## Что появилось

- `RootMenu` — компактное корневое меню.
- `RootRoute` — контракт раздела приложения.
- `QtMainWindow` — одно центральное окно без split-панелей.
- `QStackedWidget` — один активный смысловой экран.
- Lazy-loading — экран создаётся только при первом открытии.
- Верхняя панель — демо-данные, настройки, тема и выход.
- Нижняя строка — короткий статус текущего действия.

## Корневые разделы

| Видимый пункт | Полный раздел |
|---|---|
| `ПО` | Программное обеспечение |
| `ТО` | Техническое обеспечение |
| `Компоненты` | Редактор компонентов |
| `Энергия` | Электроэнергия |
| `NPV` | NPV-анализ |
| `Экспорт` | Экспорт |

Полные названия используются в заголовке экрана и tooltip, а меню остаётся коротким.

## UX-ограничения

- один активный экран в центре;
- без постоянных двигающихся панелей;
- короткие visible labels;
- пояснения только через tooltip/help;
- тяжёлые будущие экраны не создаются при старте;
- расчётная логика остаётся вне Qt-виджетов.
