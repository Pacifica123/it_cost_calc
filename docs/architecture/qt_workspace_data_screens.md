# Qt workspace data screens

Патч `qt-workspace-data-screens` переносит первый шаг маршрута ПО/ТО в Qt.

## Что добавлено

- общий presenter для областей ПО и ТО;
- отдельные Qt-экраны `SoftwareWorkspaceScreen` и `TechnicalWorkspaceScreen`;
- компактный stepper `Данные → GA → AHP → Pareto → Гибрид`;
- карточки CAPEX, OPEX, разовых затрат и количества позиций;
- раскрываемые таблицы категорий, CAPEX и OPEX.

## Ограничения

В этом патче реализован только шаг `Данные`. Шаги GA, AHP, Pareto и Hybrid
остаются отключёнными до следующих патчей миграции.

## UX-правила

- экран показывает одну смысловую задачу;
- длинные объяснения вынесены в подсказки;
- подробные таблицы скрываются в раскрываемых блоках;
- ПО и ТО используют разные профили категорий.


## Layout fix после визуальной проверки

Фикс-патч к шагу `qt-workspace-data-screens` уточняет поведение таблиц:

- таблицы данных работают в режиме чтения без лишних кнопок `Демо` и `Ещё`;
- раскрытые таблицы получают фиксированную компактную высоту;
- пустые таблицы не показывают техническую сетку;
- Qt enum-like значения в table model нормализуются для корректных заголовков;
- при наличии данных автоматически раскрываются только полезные секции.
