# Qt design system

## Назначение

Документ фиксирует первый технический слой будущего PySide6-интерфейса. Он не переносит бизнес-экраны и не меняет расчётную логику. Его задача — дать единый набор тем, виджетов и правил плотности до начала переноса экранов.

## Принципы

- видимый текст остаётся коротким;
- длинные объяснения идут в tooltip/help;
- основной стиль — flat design;
- темы мягкие и без резкого контраста;
- базовые виджеты проверяют короткие подписи;
- тяжёлые экраны не создаются на старте.

## Состав

| Файл | Назначение |
|---|---|
| `src/ui_qt/design/tokens.py` | токены тем, шрифтов и отступов |
| `src/ui_qt/design/theme_manager.py` | загрузка и применение QSS |
| `src/ui_qt/design/styles_light.qss` | мягкая светлая тема |
| `src/ui_qt/design/styles_dark.qss` | мягкая тёмная тема |
| `src/ui_qt/widgets/compact_label.py` | короткий label с проверкой длины |
| `src/ui_qt/widgets/info_hint.py` | компактная подсказка для длинных текстов |
| `src/ui_qt/widgets/action_bar.py` | панель главных и вторичных действий |
| `src/ui_qt/widgets/status_strip.py` | короткая статусная строка |
| `src/ui_qt/widgets/empty_state.py` | пустое состояние с одним действием |
| `src/ui_qt/widgets/collapsible_section.py` | скрываемый необязательный блок |
| `src/ui_qt/widgets/settings_panel.py` | компактная панель настроек |

## Правило текста

`CompactLabel`, `ActionBar`, `StatusStrip`, `EmptyState` и `CollapsibleSection` используют общий лимит: не больше 10 слов в видимом тексте. Нарушение считается ошибкой проектирования нового UI.

## Текущий статус

Дизайн-система подключена к минимальному Qt shell. Основной Tkinter-интерфейс остаётся рабочим и не переключается этим патчем.
